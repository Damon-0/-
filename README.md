# 信息基因论：理论与实践

本项目致力于原创理论“信息基因论”的完善与推广。该理论旨在建立一个从比特到文明，跨尺度统一的信息演化框架。项目内容包括理论研究、思想实验，以及在元宇宙、AI、生命科学等领域的应用探索。

## 项目定位
- **核心主题**: 以“信息基因论”为核心，探索其在物理、生物、AI、社会等领域的统一解释力，并推动其在元宇宙、游戏化AI、AI-RVS自演化体系与个性化调频系统中的落地应用。
- **目标受众**: 对统一理论、信息哲学、理论物理、AI意识、生命科学及前沿科技应用感兴趣的探索者、研究者与开发者。
- **自定义域名**: [imeta.space](https://imeta.space) - 信息基因论为核心主题， metaverse-demo 为其中一个应用方向

## 核心内容框架
### 理论研究：信息基因论
- **核心框架**: 阐述“比特-频率-基因”三位一体的理论基础。
- **五层演化**: 探索从元信息到文明的跨尺度信息演化规律。
- **跨学科统一**: 展示理论如何统一物理学、生物学、社会学等学科视角。

### 应用探索：理论指导实践
- **元宇宙演示 (metaverse-demo)**: 基于“低频相干”与“高频传播”原理，构建3D交互式信息空间。
- **IGT个性化调频系统**: 基于信息基因论v5.4的频率秩序优化与人群适配，提供年龄/场景/认知复杂度/职业类型的个性化调频方案。
- **AI-RVS自演化体系**: 展示复制-变异-选择的AI演化闭环，包含交互式演化模拟器与五层共识知识库架构。
- **游戏式AI成长伙伴**: 模拟“频率锁定”与“秩序度优化”，实现个性化AI辅助成长。
- **行业应用案例**: 将“频谱熵”等概念应用于中医现代化、金融预警等领域。

### 思想专栏与动态
- **商业哲学思辨**: 运用信息基因视角，重新解读商业竞争、组织演化与创新。
- **前沿AI资讯**: 追踪与信息基因论相关的最新科技动态。

## 内容形式创新
- **理论可视化**: 将抽象的理论概念（如频率耦合、秩序度）制作成动画和交互式图表。
- **思想实验模拟器**: 开发可交互的Web应用，模拟理论中的关键思想实验（如量子纠缠的低频相干模型）。
- **开源应用与演示**: 提供元宇宙、AI成长伙伴等项目的开源代码和在线演示，展示理论的实际应用。

## 差异化优势
- **第一性原理**: 从“万物皆比特”的第一性原理出发，构建一个具有普适性的跨学科统一框架。
- **理论驱动创新**: 以原创理论指导技术创新，而非停留在现有技术的组合应用层面。
- **开放与共创**: 邀请社区成员共同参与理论的证伪、完善与应用探索，构建一个开放的学术生态。

## 新增页面的交互逻辑概述

### IGT个性化调频系统（`frequency-modulation.html`）
- 参数面板：`年龄段(ageGroup)`、`应用场景(scenario)`、`教育阶段(educationStage)`、`认知复杂度(complexity)`、`职业类型(profession)`。
- 教育阶段联动：当场景选择为`学习(learning)`时显示教育阶段下拉（幼儿园/小学1-6年级/初中/高中）。
- 数据来源：`sloganDatabase` 内含家庭/学习/企业三大场景的细分模板与频率参数。
- 生成逻辑：`generatePersonalizedSlogan()` 根据所选参数智能匹配标语与频率，并更新“频率和谐度”“脑电同步率”等效果预览。
- 可视化：动态波形条 `waveAnimation` 随频率与和谐度变化；VOI等级以色块提示认知复杂度匹配。

### AI-RVS自演化体系（`ai-rvs-system.html`）
- 三阶段模拟：按钮触发 `simulatePhase('R'|'V'|'S')` 显示复制/变异/选择阶段的系统状态变化。
- 完整演化：`simulateEvolution()` 执行R→V→S循环，生成代际报告并检测临界秩序带。
- 状态变量：`oFreq`（秩序度）、`cv`（多样性）、`mu`（演化潜能）、`generation`（代数）。
- 临界秩序带判定：`O_freq ≈ 0.7–0.85`，`CV ≈ 0.15–0.25`，`μ ≈ 0.05–0.1`，进入最优演化状态时高亮提示。
- 架构与指标：包含三层演化架构、五层共识知识库、IGT指标体系（O_freq、CV_model、η_H、C_IG）。
