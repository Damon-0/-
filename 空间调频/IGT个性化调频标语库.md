# IGT个性化调频标语库
*基于信息基因论v5.4的年龄段与场景适配*

## 理论基础

### 信息基因论v5.4核心原理
- **频率秩序度公式**：Ofreq = 1 - Sfreq/log₂N
- **RVS机制**：复制-变异-选择三要素
- **三频一致性**：语义频率 + 视觉频率 + 声学频率
- **VOI等级**：VOI1-VOI5，对应不同认知复杂度

### 年龄段认知特征
| 年龄段 | 主导脑波 | 认知特点 | VOI适配 |
|--------|----------|----------|---------|
| 儿童(0-12岁) | θ波(4-7Hz) | 具象思维、情绪敏感 | VOI1-VOI2 |
| 青少年(13-18岁) | α波(8-13Hz) | 抽象思维发展、情绪波动 | VOI2-VOI3 |
| 成年人(19-60岁) | β波(14-30Hz) | 逻辑分析、目标导向 | VOI3-VOI4 |
| 老年人(60岁+) | θ波回升 | 经验整合、情感回归 | VOI2-VOI3 |

## 四、使用指南与效果评估

### 4.1 个性化选择算法

#### 年龄-场景匹配公式
```
个性化VOI = 基础VOI + 场景调整系数 + 个体差异修正

其中：
- 场景调整系数：家庭(+0.2)，学习(+0.3)，企业(-0.1)
- 个体差异修正：通过前期测评确定(-1到+1)
```

#### 频率调谐步骤
1. **基线测量**：使用脑电设备测量个体基础频率
2. **VOI选择**：根据年龄、场景、个体特征选择合适VOI等级
3. **标语定制**：从对应库中选择最匹配的标语内容
4. **效果监测**：定期评估调频效果，动态调整

### 4.2 效果评估指标

#### 客观指标
- **脑电同步率**：目标频率与实际频率的匹配度 (目标>80%)
- **认知表现**：注意力、记忆力、创造力测试分数提升 (目标>15%)
- **情绪状态**：积极情绪比例增加 (目标>20%)

#### 主观指标
